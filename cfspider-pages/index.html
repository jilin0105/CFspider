重塑数据抓取的核心特性

海量边缘代理 IP 池

基于 Cloudflare Workers 构建，覆盖全球 300+ 节点，支持 VLESS/HTTP 双模式。每次请求动态切换 IP，彻底解决 IP 封禁难题。

CloakBrowser 深度反检测

C++ 源码级 Chromium 补丁，提供真实 TLS 指纹与 UA。一行代码开启 `stealth=True`，绕过极验、Turnstile、5s 盾等顶级防护。

AI 智能浏览器

内置 cfspider-browser（Electron+React），支持自然语言指令控制。像真人一样自动浏览、搜索、点击，内置多模型视觉分析。

高性能异步并发

原生支持 asyncio，内置 StealthSession 连接池管理。海量并发请求下依然保持极低的资源占用和毫秒级响应速度。

优雅，仅需一行代码

告别繁琐的 Headers 伪造、Cookie 管理和重试逻辑。CFspider 将最复杂的反检测机制封装成了最简单的 API。

  • ✔️ 自动维护请求 Session
  • ✔️ 智能通过 TLS 指纹校验
  • ✔️ 无头与有头模式无缝切换
import cfspider

# HTTP 级隐身：真实 TLS 指纹 + UA
r = cfspider.get("https://protected-api.com", stealth=True)
print(r.json())

# 浏览器级隐身：执行 JS，绕过验证码
r = cfspider.get("https://secure-site.com", browser=True)
print(r.text)

# 配合 Workers 代理池使用
r = cfspider.get("https://example.com", 
                 cf_proxies="https://my-workers.dev", 
                 stealth=True)
